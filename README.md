# dotfiles

My MacOS focused development environment dotfiles.

## Installation

Link dotbot paths and run `brew bundle --file .Brewfile`

## Applications

+ Yabai WM & SKHD
+ Homebrew
+ ZSH
  - FIsh-esque history search
  - Autocomplete
  - Vim mode
+ Kitty
+ Starship Prompt
+ Tmux
+ Neovim
  - Ruby development focused
  - LSP
  - Treesitter
+ LazyGit
+ Wallpaper - Alena Aenami
